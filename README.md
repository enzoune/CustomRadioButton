# 🎓 Interactive Course Selection Card (Custom Radio UI)

> Um componente de interface moderno e responsivo para seleção de planos de assinatura, desenvolvido puramente com HTML5 e CSS3, utilizando lógica de estado nativa.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo principal foi criar um seletor de cursos interativo onde o usuário pode escolher entre diferentes tecnologias (HTML & CSS, JavaScript, PHP & MySQL) de forma visualmente atraente.

A implementação foca em transformar elementos de formulário padrão em componentes de UI ricos, garantindo que a escolha do plano seja intuitiva e fluida para o usuário final.

## ✨ Destaques e Diferenciais Técnicos

* **Gerenciamento de Estado Nativo:** Uso eficiente de `input type="radio"` ocultos e labels vinculadas via atributo `for`, permitindo o controle da interface sem a necessidade de JavaScript.
* **Arquitetura Modular:** Organização em classes lógicas como `.card`, `.content` e `.box`, facilitando a manutenção e a adição de novos planos ou categorias de curso.
* **Hierarquia Visual:** Estrutura de conteúdo que separa claramente o título do curso (`.subject`) do valor da assinatura (`.price`), otimizando a leitura e a tomada de decisão.
* **Design Centrado no Usuário:** Substituição do botão de rádio padrão por indicadores visuais personalizados (`.circle`), proporcionando um visual profissional e moderno.
* **Organização Profissional:** Separação estrita entre a estrutura semântica (`index.html`) e a identidade visual (`css/style.css`), seguindo os padrões da engenharia de software.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e lógica de seleção de formulário.
* **CSS3:** Estilização de estados (hover e checked), gerenciamento de layout e design responsivo.
